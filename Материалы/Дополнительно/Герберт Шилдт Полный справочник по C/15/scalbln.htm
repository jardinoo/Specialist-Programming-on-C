Функция scalbln

#include <math.h>
float scalblnf(float val, long int exp);
double scalbln(double val, long int exp);
long double scalblnl(long double val, long int exp);

Функции scalbln(), scalblnf() и scalblnl() добавлены в версии С99.

Каждая функция семейства scalbln() возвращает произведение параметра val и значения FLT_RADIX, возведенного в степень, которая равна значению параметра ехр. т.е. в результате получается val * FLT_RADIXexp

Макрос FLT_RADIX определен в заголовке <float.h>, и его значение равно основанию системы счисления, используемой для представления вещественных чисел.
